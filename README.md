## Project Overview

This project demonstrates the fundamentals of **AI Recommendation Logic** using **content-based filtering**.
The system:
- Accepts user interests as input
- Converts text into numerical vectors using TF-IDF
- Measures similarity using Cosine Similarity
- Recommends the most relevant item
---
## Features
✔ User preference input  
✔ Content-based recommendation system  
✔ TF-IDF text vectorization  
✔ Cosine similarity matching  
✔ Simple and beginner-friendly implementation

## Technologies Used
- Python
- Scikit-Learn
---
---
## How It Works
1. The user enters their interests.
2. The program stores predefined item descriptions.
3. TF-IDF converts text data into numerical vectors.
4. Cosine Similarity compares the user's interests with available items.
5. The system returns the best recommendation.
